# Matemática Computacional em Python

Este repositório apresenta a conversão dos conceitos de **matemática computacional** em códigos Python, conforme o conteúdo ensinado em sala de aula na graduação.

---

## Estrutura do Projeto

| Tópico                               | Conteúdo Principal                                                         |
|---------------------------------------|---------------------------------------------------------------------------|
| **Aritmética Básica**                 | Operações básicas, propriedades, algoritmos de cálculo                     |
| **Álgebra Linear Computacional**      | Matrizes, vetores, operações entre matrizes, resolução de sistemas         |
| **Funções Matemáticas**               | Implementação de funções, gráficos e análise de resultados                 |
| **Equações e Sistemas Lineares**      | Métodos de resolução, decomposição, substituição, aplicação                |
| **Métodos Numéricos**                 | Solução algorítmica, aproximações, erros, métodos de Newton e Bissecção    |
| **Cálculo Computacional**             | Derivadas, integrais, aplicações com algoritmos                            |
| **Exercícios/Projetos**               | Exemplos práticos e exercícios resolvidos                                  |
